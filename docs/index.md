# Hasher v0.1

Hasher is a lightweight Python library providing implementations of **Argon2**, **SHA-256**, and **BLAKE3** algorithms for secure and efficient hashing.

## Features
- Modular algorithm structure
- Configurable parameters for Argon2
- Easy-to-extend architecture
- Example scripts for testing

## Directory Structure
- `hasher/` — Core implementation
- `tests/` — Unit tests for all algorithms
- `examples/` — Example scripts for demonstration
